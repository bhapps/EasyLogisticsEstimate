About Easy Logistics Estimate

Standalone web-browser Tool used for estimating the logistical supply chain requirements (costs, packaging, delivery etc) from data extracted from SAP using the Ctrl+Y copy/paste function in SAP

Lines, Materials & QTY

In SAP, use Ctrl+Y to select the lines for processing and ensure only to select the Line, Material and QTY columns only.

Once copied, paste these into the field below for processing and compile the estimates.

You can load basic example, large example or error example

What is this?

Estimates

Your estimations will appear here once processing is completed and validated. Any invalid lines will be displayed accordingly.
